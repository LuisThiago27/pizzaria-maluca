6 - Método de Login (Parte 3)
 
Problema com o MySql (localhost)

A aplicação exibe o seguinte erro no terminal do Visual Code:

ER_NOT_SUPPORTED_AUTH_MODE: Client does not support authentication protocol requested by server; consider upgrading MySQL client


SOLUÇÃO:
Faça o comando dentro do MySql (em uma página em branco normal):

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'password';

Obs:
'root' = colocar o nome do usuário do banco.
'password' = colocar a senha do usuário informado.

Depois:

flush privileges;
